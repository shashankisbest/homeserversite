When you SSH into your server, you're working on the server's operating system, not your Windows laptop. Your server doesn't automatically have Django installed just because your laptop does.

Check first:

python3 --version
pip3 --version

Then check if Django exists:

django-admin --version

or

python3 -m django --version

If you get "command not found" or "No module named django", install it.

For a Docker-based project, the usual workflow is:

mkdir myproject
cd myproject

python3 -m venv venv
source venv/bin/activate

pip install django

Now:

django-admin startproject website .

python manage.py startapp filetransfer

create app accounts
in models.py:
    from django.db import models

    class User(models.Model):
        username = models.CharField(max_length=50, unique=True)
        password = models.CharField(max_length=255)
        is_admin = models.BooleanField(default=False)

        def __str__(self):
            return self.username
